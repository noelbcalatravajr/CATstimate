ℹ How to use ArcRise ▼
  1. Add one segment per continuous run — a stair flight, a ramp, or a landing.
  2. Set the number of handrail lines (top rail, mid rail, etc.) — each line has its own arc length.
  3. Enter each line's horizontal arc length.
  4. Pick an elevation method — RL values, slope ratio, slope angle, fall (1:n), step count, or an assumed slope.
  5. Results update live. Net total + wastage and a per-line breakdown appear at the bottom.
Inclined length = √(arc² + rise²). End allowance is added to both ends of every line.
